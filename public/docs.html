API Documentation

Welcome to the REST API. Below is the list of available endpoints.

Users

GET /api/users

List all users. Supports pagination (limit, offset), search (name, email), and sorting.

Query Params: limit, offset, search, sort, order

GET /api/users/:id

Get a specific user by ID.

POST /api/users

Create a new user. Requires name and email. Phone optional (+32 4xx...).

PUT /api/users/:id

Update a user. Requires name and email.

DELETE /api/users/:id

Delete a user.

News Posts

GET /api/news

List news posts. Supports pagination, search, and sorting.

GET /api/news/:id

Get a specific news post.

POST /api/news

Create a news post. Requires title, content, and user_id.

PUT /api/news/:id

Update a news post.

DELETE /api/news/:id

Delete a news post.

Extra Features & Testing

1. Search functionality

Filter users by name.

GET /api/users?search=...

2. Pagination

Limit the number of results per page.

GET /api/users?limit=...

3. Sorting

Sort users by name (A-Z).

GET /api/users?sort=name
